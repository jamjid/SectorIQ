# SectorIQ
- Build for Kaggle 5-day Agent Camp Hackthon.
- Brief introduction: 
      A multi-agent system that autonomously researches industries, synthesizes executive briefings, and creates interactive assessments to accelerate employee onboarding from days to minutes.
- Since the winners in this hackthon will be announced before the end of December 2025, this repository just shows source code of the web app as you can see in py file named app. The whole project will be available in 2026.
